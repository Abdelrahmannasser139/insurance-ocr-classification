# Project Structure

## ├── notebook.ipynb
## ├── ocr_insurance_dataset.pkl
## ├── README.md
## └── requirements.txt

This folder contains the main components of the insurance OCR classification project, including:

- **notebook.ipynb**: The Jupyter Notebook for analysis and model training.
- **ocr_insurance_dataset.pkl**: The serialized dataset used for training the OCR model.
- **README.md**: This file, providing an overview of the project structure.
- **requirements.txt**: A list of the Python packages required to run the project.

## Overview

The purpose of this project is to classify insurance-related documents using OCR technologies. 

### Getting Started

1. Clone the repository.
2. Install the dependencies from `requirements.txt`.
3. Run the Jupyter Notebook to start the analysis.
